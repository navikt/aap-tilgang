## AAP oppslag
Applikasjon for å sjekke om brukere har tilgang til sak/ressurs

## Komme i gang
Bruker gradle wrapper, så bare klon og kjør `./gradlew build`

## Henvendelser
Spørsmål knyttet til koden eller prosjektet kan stilles som issues her på GitHub
